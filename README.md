# AI 赛事库 · AI Competitions Hub

一个由飞书多维表格自动生成的**公开赛事目录**静态网站，汇集 AI 赛事、AI 电影节与 AI 设计类比赛。

## 收录内容

数据来自飞书多维表格，共 **1,346** 条赛事：

| 分类 | 数量 |
| --- | --- |
| AI 赛事收集 | 102 |
| AI 电影节收集 | 220 |
| AI 设计类收集 | 1024 |

每条赛事包含：赛事名称、类型、地区、截止日期、倒计时、奖金/奖项、明年预期时间、官网链接、说明、参赛资格。

## 功能

- 🔍 关键词搜索（名称 / 类型 / 地区 / 说明 / 官网）
- 🗂️ 多维筛选：大类 · 状态（进行中/已截止/待公布）· 类型 · 地区
- 🔗 官网直达（卡片与详情页均带报名/展示入口）
- 🖼️ 展览画布（可拖拽、缩放的赛事墙）

## 本地预览

```bash
python -m http.server 8000
# 打开 http://127.0.0.1:8000/
```

## 文件结构

- `index.html` — 页面结构
- `styles.css` — 纸纹主题样式
- `app.js` — 检索 / 筛选 / 详情 / 画布逻辑
- `data.js` — 赛事数据（`window.EVENT_DATA`）

> 数据快照生成于飞书多维表格，后续可重新抽取更新 `data.js`。
